# UMA基础示例：实时数据可视化

## 项目概述
一个简单的UMA项目，展示如何将实时数据流转化为动态可视化作品。

## UMA概念体现
- **UMA-001 泛在性**：使用云端API获取实时数据
- **UMA-103 规模即伦理**：处理大规模数据流
- **UMA-402 数据河流**：将数据流转化为美学体验
- **UMA-302 无缝沉浸**：流畅的动态可视化

## 技术栈
- **前端**：HTML5 Canvas + JavaScript
- **数据源**：公共API（天气、金融、社交媒体）
- **部署**：静态托管（GitHub Pages / Vercel）

## 文件结构
```
uma-basic/
├── index.html          # 主页面
├── style.css          # 样式
├── app.js             # 主逻辑
├── data-fetcher.js    # 数据获取
├── visualizer.js      # 可视化引擎
└── config.json        # 配置文件
```

## 快速开始
1. **克隆或下载本项目**
2. **在浏览器中打开 index.html**
3. **修改 config.json 选择数据源**
